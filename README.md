## ğŸš€ Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/neon-snake.git

```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

### Building for Production

```bash

npm start
```
